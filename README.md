# NotchMusic

A beautiful, minimal Spotify player that lives in your MacBook Pro's notch.

![NotchMusic Demo](https://via.placeholder.com/400x200?text=NotchMusic)

## Features

- **Perfect Integration**: Sits cleanly inside the hardware notch
- **Interactive**: Click to expand for full track info and controls
- **Dynamic**: Real-time Spotify sync with extracting dominant album colors
- **Optimized**: 0% background CPU usage, battery-friendly, zero polling

## Requirements

- macOS 13.0+
- MacBook Pro with a notch (2021+)
- Spotify desktop app

## Installation

1. Open `NotchMusic.xcodeproj` in Xcode.
2. Press `Cmd + R` to build and run.
3. macOS will ask for permission to control Spotify. Click **OK**.

*(To install permanently: Build the app, find `NotchMusic.app` in the Products folder, and drag it to your Applications folder).*
